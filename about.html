About me!

Hey! I'm Tristan, I'm currently doing the Computer Systems Technician - Network and Cloud Technologies program at Niagara College. My interest in computers started when I was a kid and got my own computer for the first time. I spent a lot of time on my computer, upgrading it, playing games, and coding! In early 2024 I started using Linux as an operating system, and later as my hypervisor (Proxmox) for my homelab. My homelab is my pride and joy and my current largest server deployment, running various services including but not limited to: TrueNAS, OPNsense, Immich, Docker, and Pihole.
